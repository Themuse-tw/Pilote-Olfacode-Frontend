☁ Backend cloud (recommandé)

Synchronise tes données entre PC et téléphone, conserve un historique pérenne en base Postgres, et active la synchro automatique PhantomBuster. Sans backend, l'app reste fonctionnelle en mode 100% local (localStorage).

🔑 Clé API OpenAI (mode local uniquement)

Si tu utilises le backend cloud, la clé OpenAI est stockée côté serveur (plus sécurisé). Cette saisie est utilisée uniquement en mode local. Récupère ta clé sur platform.openai.com/api-keys. Conseil : fixe une limite de dépense mensuelle (ex: 30 $) sur ton compte OpenAI.

👻 PhantomBuster — agents à synchroniser (nécessite backend)

Liste les IDs de tes agents PhantomBuster (un par ligne). À chaque clic sur "⟳ Sync PhantomBuster" dans l'onglet Prospects, le backend récupère leurs derniers outputs et insère les nouveaux profils. La clé API PhantomBuster est sur Render (env var PHANTOMBUSTER_API_KEY).

Où trouver un ID d'agent : phantombuster.com/dashboard → clic sur ton agent → l'URL contient /agents/<ID>
